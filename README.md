# 思源日历

一个简洁开源的日历插件。

### 代办：

##### 基础界面

- [x] 基本日历界面，粒度为30min，可以选择日、周视图，有左右切换、和“回到今天”按钮。
- [x] 在日历界面上选中时段，弹出创建日程功能，创建界面包括：

  1. 日程名称
  2. 备注
  3. 自定义颜色“6位hex码”
  4. 从属思源笔记文档（可以在这个界面输入名称搜索思源文档，会显示下拉列表，可以选中）s
- [] 可以双击把日程标记为已完成，再次双击取消。
- [] 拖动已有的日程，可以挪动它的位置

后端：

- [x] 用一个json储存日历数据，每个日程包含以下变量：

  1. 开始时间
  2. 结束时间
  3. 是否已完成
  4. 颜色
